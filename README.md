# Pixel Scaling Software

## General Description
This first model of the program uses Python's Pillow library + terminal commands to scale pixel-art from 8/16/32/64 bits up to 640x640

## Usage:
You need the Pillow library from Python:
pip install Pillow

After running the program, a terminal should pop-up.
Run "python pixel_scaling.py pixel_art.png 4" with your pixel-art inside the same folder as the program.
